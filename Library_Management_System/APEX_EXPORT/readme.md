#  Oracle APEX Export

The complete Oracle APEX application export file is included in this project.

## Export File

```text
APEX_Export/f289183.sql
```

## What It Contains

- All application pages
- Interactive reports
- Dashboard components
- Navigation menus
- Authorization settings
- SQL queries
- UI themes and styling
- Application logic

## How to Import

1. Open Oracle APEX
2. Go to App Builder
3. Click Import
4. Upload the `f289183.sql` file
5. Follow the import wizard
6. Run the application
